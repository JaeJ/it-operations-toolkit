# Active Directory Module

## Overview

The Active Directory module provides user, group, and privileged account reporting capabilities for the IT Operations Toolkit.

The goal of this module is to support identity administration, auditing, reporting, and operational visibility within Windows environments.

---

## Business Problem

Active Directory environments often contain thousands of users, groups, and administrative accounts.

Administrators need efficient ways to audit directory objects, identify privileged access, and generate operational reports.

This module provides reusable tools to improve visibility and support administrative decision-making.

---

## Current Scope

### User Reporting

- User Inventory
- Account Status Reviews

### Group Reporting

- Group Inventory
- Membership Reviews

### Privileged Access Reviews

- Administrative Account Reporting
- Privileged Group Audits
