# Azure Module

## Overview

The Azure module provides cloud inventory, virtual machine reporting, storage reporting, and Azure administration capabilities for the IT Operations Toolkit.

The goal of this module is to support cloud operations, resource visibility, governance, and operational reporting.

---

## Current Scope

### Inventory

- Azure Resource Inventory
- Resource Discovery
- Subscription Visibility

### Virtual Machines

- VM Reporting
- VM Status Reviews
- Operational Visibility

### Storage

- Storage Account Reporting
- Capacity Visibility
- Resource Tracking
