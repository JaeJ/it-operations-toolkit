# Microsoft 365 Module

## Overview

The Microsoft 365 module provides user, licensing, and mailbox reporting capabilities for the IT Operations Toolkit.

The goal of this module is to support tenant administration, operational reporting, and visibility into Microsoft 365 environments.

---

## Current Scope

### User Reporting

- User Inventory
- Account Status Reviews

### License Reporting

- License Consumption
- SKU Visibility

### Mailbox Reporting

- Mailbox Inventory
- Recipient Reporting
