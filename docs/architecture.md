# IT Operations Toolkit Architecture

## Overview

The IT Operations Toolkit is an enterprise-focused portfolio project designed to demonstrate infrastructure engineering, systems administration, automation, reporting, troubleshooting, and cloud operations skills.

The toolkit consolidates capabilities from:

- PowerShell Automation
- Infrastructure Scripts
- Azure Lab

into a unified operational platform.

---

## Core Modules

### Active Directory

User audits, group reporting, permissions analysis, and administrative reviews.

### Azure

Resource inventory, cloud administration, reporting, cleanup operations, and governance.

### Diagnostics

System health checks, service analysis, event log review, and error investigation.

### Microsoft 365

Tenant reporting, licensing reviews, mailbox auditing, and user administration.

### Networking

DNS diagnostics, connectivity testing, network analysis, and troubleshooting.

### Reporting

CSV exports, JSON exports, HTML reports, dashboards, and scheduled reporting.
